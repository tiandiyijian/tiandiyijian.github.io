快速排序

发表于 2019-08-05 | 分类于 leetcode

快速排序又称划分交换排序，简称快排，它与归并排序算法一样，也是分治策略的典型应用，但二者之间也有本质区别。归并排序的计算量主要消耗于有序子向量的归并操作，而子向量的划分几乎不消耗时间（就是对半分）；快排正好相反，它可以在$O(1)$时间内，由子问题的解直接得到原问题的解，但为了将原问题划分为两个子问题，却需要$O(n)$的时间（选取轴点）。

所谓轴点（pivot），就是左侧元素均不比它大，右侧元素均不比它小的点。如图所示：

快排的思路就是，先选取一个轴点（轴点选取好之后它就不会再变动了，即这个位置上的元素就是最终排好序的数组中该位置上的元素，可以看成这个点已经是排好序的了），然后把这个数组分为轴点左右两边两个子数组，再分别对它们进行快排，注意到最终排序好的数组中轴点左/右边的成员和当前可能无序的左/右子数组中的成员是相同的，即现在轴点右边的元素不会跑到轴点左边去，左边的也不会跑到右边去。所以这两个子数组的排序可以独立进行，更重要的是，一旦它们都完成排序，即可立即（在$O(1)$时间内）得到整个数组的排序结果。排序的代码为：

Multi-label Image Recognition by Recurrently Discovering Attentional Regions

发表于 2019-07-17

摘要

多标签图像识别是计算机视觉中基础但实用的问题，因为真是世界的图像总是会包含丰富多样的语义信息。目前解决这个问题的方法大多依赖于额外的步骤来产生假定区域，即图片中要识别目标出现的区域，这就导致了多余的计算并且达不到最优的效果。作者提出了一种循环记忆-注意模块从而实现了多标签图像分类的可解释性和上下文有关性。作者提出的这种方法在准确率和性能方面的表现都优于 state-of-the-arts。

介绍

最近，卷积神经网络通过强有效的图片特征提取在视觉识别识别/分类任务上取得了巨大的成功，于是通过结合一些目标定位技术卷积神经网络也被应用到了多标签图像识别问题上。这些方法通常可以分为两步：

  • 产生一系列包含所有可能的前景目标的假定区域
  • 训练分类器或神经网络来预测假定区域的标签的 score ，最后把所有假定区域的预测结果汇总起来得到多标签分类结果

这些方法在产生假定区域时进行了大量冗余的计算并且没有充分考虑到前景目标之间的依赖关系从而导致在面对复杂场景时得不到最优的表现。

最近，Wang等人提出通过将循环神经网络和卷积神经网络结合来共同表征语义标签依赖性和图像标签相关性。然而，他们的模型忽视了语义标签和图像内容之间的显式关联，并且没有充分利用图像中的空间背景。

基于这些情况，作者提出了一种端到端的可训练的框架，该框架明确地发现与多个语义标签相对应的图像尺度上的注意区域，并从全局角度捕获这些区域间的上下文依赖性。

注意到，作者提出的方法并不需要提取假定区域的步骤，而是提取注意区域，这是对之前的基于 object proposal 的方法的一大改进，主要体现在减少计算量方面。因为obiect proposal方法需要大量的计算来提取假定区域，而作者提出的方法会自动发现K个注意区域，然后会根据这K个区域来预测标签。后面会详细介绍。

系统分析项目演示视频

发表于 2019-06-23 | 分类于 系统分析

使用说明：

用户首次使用时需要点击登录进行微信授权，授权过一次之后以后可以自动登录了。

每个用户注册即送30余额，在“我的”页面，用户可以点击“我的余额”查看自己的余额；用户可以分别点击“发布问卷”和“发布简历”来发布问卷和简历供别的用户填写，在发布问卷或简历的时候不仅要创建问题，也要填入别人填写之后能得到的报酬，当别人填写自己发布的问卷或简历时，将会从自己的余额中扣除相应的钱。

用户能在问卷和简历页面分别看到别人发布的问卷和简历，也可以点击上方的搜索框搜索自己感兴趣的内容，在成功填写并提交之后会得到相应的报酬，每个问卷和奖励用户只能填写一次，发布的人自己也可以填写。

在“我的”界面分别点击“我的问卷”或“我的简历”可以进入查看自己发布的问卷或简历列表页面，点击其中一项之后可以进入回答该问卷或简历的用户名列表界面，点击其中一项可以进入该用户的回答页面；点击上方的停止回答按钮，别的用户将不可以再回答该问卷或简历并获得报酬。
