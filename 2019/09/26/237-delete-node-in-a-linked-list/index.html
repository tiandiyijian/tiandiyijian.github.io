分析

常规思路是从头开始扫描找到这个节点的前一个节点，但是这道题并没有给出要删除链表的头结点，只给出了要删除的那个节点，刚开始想这不就没办法找到要删除的节点的前一个节点了吗。但是其实没有这样做的必要（也没办法这样做了哈哈），就比如说例1，要删除第二个节点5，我们可以让那个节点的值等于它后面的那一个节点的值1，然后再把它指向它的下一个节点的下一个节点9，这不就相当于删除了节点5吗。
