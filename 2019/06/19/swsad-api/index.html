搜索简历

基本信息

Path： /searchResume

Method： POST

接口描述：

根据用户的输入搜索简历

请求参数

Headers

参数名称 参数值 是否必须 示例 备注
Content-Type application/x-www-form-urlencoded 是

Body

参数名称 参数类型 是否必须 示例 备注
openid text 是 用户的openid
value text 是 用户在搜索框的输入

返回数据

名称类型是否必须默认值备注其他信息
object []非必须

item 类型: object

├─ idnumber必须简历id
├─ titlestring必须简历名
├─ descriptionstring必须简历描述
├─ idOfReleaserstring必须简历发布者的openid
├─ rewardstring必须回答简历可以获得的奖励
├─ questionsobject []必须简历的问题

item 类型: object

├─ titlestring必须问题名
├─ answerstring必须问题的回答，初始为null

搜索问卷

基本信息

Path： /searchSurvey

Method： POST

接口描述：

根据用户的输入搜索问卷

请求参数

Headers

参数名称 参数值 是否必须 示例 备注
Content-Type application/x-www-form-urlencoded 是

Body

参数名称 参数类型 是否必须 示例 备注
openid text 是 用户的openid
value text 是 用户在搜索框的输入

返回数据

名称类型是否必须默认值备注其他信息
object []非必须

item 类型: object

├─ idnumber必须问卷id
├─ titlestring必须问卷名
├─ descriptionstring必须问卷描述
├─ numOfQuestionsnumber必须问卷问题数
├─ idOfReleaserstring必须问卷发布者的openid
├─ rewardstring必须回答问卷可以获得的奖励
├─ questionsobject []必须问卷的问题

item 类型: object

├─ radioboolean必须是否单选
├─ titlestring必须问题名
├─ choicesstring []必须问题选项

item 类型: string

├─ 非必须A、B、C、D四个选项
├─ currentstring必须已选选项，初始为null

登录

基本信息

Path： /login

Method： POST

接口描述：

小程序端向微信服务器请求code发送给服务端，然后服务端向微信服务器获取该用户的openid，并在数据库为初次使用的用户添加信息，最后把openid返回给小程序

请求参数

Headers

参数名称 参数值 是否必须 示例 备注
Content-Type application/x-www-form-urlencoded 是

Body

参数名称 参数类型 是否必须 示例 备注
code text 是 小程序wx.login()返回的code

返回数据

名称类型是否必须默认值备注其他信息
openidstring必须用户的openid

Method： POST

接口描述：

获取某个简历收到的答案的列表

请求参数

Headers

参数名称 参数值 是否必须 示例 备注
Content-Type application/x-www-form-urlencoded 是

Body

参数名称 参数类型 是否必须 示例 备注
idOfResume text 是 某一简历的id

返回数据

名称类型是否必须默认值备注其他信息

item 类型: object

├─ questionsobject必须问题列表
├─ titlestring必须问题名
├─ answerstring必须问题答案
├─ idstring必须该答案的id
├─ imgSrcstring必须用户回答简历时上传的图片URL
