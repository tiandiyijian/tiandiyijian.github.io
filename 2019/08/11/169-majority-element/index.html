分析

题目要求求出给定数组中的众数，也就是出现次数大于$\frac{n}{2}$的数。我刚开始想到的解法是用一个字典把数组中数字出现的次数都保存下来，但是这样消耗了太多的空间。后来了解到有一个算法叫做Boyer–Moore majority vote algorithm)，它的步骤如下：

  • 初始化一个候选元素 candidate 和 计数器 counter = 0
  • 对数组中每一个元素 x：
    • 如果 counter = 0，那么 candidate = x
    • 如果 candidate = x，counter += 1，否则 counter -= 1
  • candidate 即为所求

感觉直观上不是很容易想出来，举个例子就容易看明白了：就比如说数组「1 3 2 3 3」，第一次循环 candidate = 1， counter = 1；第二次循环 candidate = 1， counter = 0；第三次循环 candidate = 2， counter = 1；第四次循环 candidate = 2， counter = 0；最后一次循环循环 candidate = 3， counter = 1；最终 candidate = 3。我最后是这么理解的，在扫描数组的过程中，肯定会遇见真正的众数，而众数的数量是大于数组长度的一半的，所以最终的 candidate 肯定为众数并且 counter 就等于众数的个数减去非众数的个数。
