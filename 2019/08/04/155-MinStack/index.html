Leetcode155-最小栈

发表于 2019-08-04 | 分类于 leetcode
题目描述

Design a stack that supports push, pop, top, and retrieving the minimum element in constant time.

  • push(x) — Push element x onto stack.
  • pop() — Removes the element on top of the stack.
  • top() — Get the top element.
  • getMin() — Retrieve the minimum element in the stack.
分析

和一般的栈的区别就是要能在常数时间内返回栈中元素最小值，有很多种方法，最常见的就是使用两个栈，一个保存元素称为元素栈，另一个保存最小值称为最小值栈。保存最小值也有两种做法，一种是在每一个元素插入后都在最小值栈中插入此时对应的最小值，这样两个栈的大小是一样的；另一种是只有在最小值发生变化时才插入或删除最小值栈中的元素，这样两个栈的大小是不一样的。这两种做法在插入、删除和获取最小值值时都会有一些细微的差别。我的做法是只用一个栈，但是这个栈中的元素是一个当前元素和它对应的最小值的 pair ，获取最小值的时候只需要访问栈尾的 pair 的第二个元素即可。代码如下：
